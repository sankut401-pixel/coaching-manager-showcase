# Coaching Manager (Showcase)

A coaching institute management app: offline-first Android (Flutter) + hosted backend (Django REST + PostgreSQL).

This repository is a *public showcase* only.
- No source code
- No APK
- No secrets

## Highlights
- Students: profiles, batch/class, insights
- Attendance: daily marking + risk list
- Fees: pending tracking + payment entry UX + reminders
- Tests: create tests, enter marks, performance trend
- Reports: CSV/TXT exports + advanced summaries
- Sync: local SQLite + push/pull sync to server

## Tech Stack
- Mobile: Flutter (Provider, sqflite)
- Backend: Django REST Framework, JWT auth, PostgreSQL
- Hosting: Render
